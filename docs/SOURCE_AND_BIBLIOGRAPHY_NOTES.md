# Source And Bibliography Notes

The bibliography provides standard background and related-work orientation for Coxeter groups, finite posets, linear extensions, order polytopes, quotient-like Coxeter constructions, permutation-set transitivity, and nearby type-A poset-cone work.

The paper proofs are written in the paper body. Bibliography entries are not hidden proof dependencies for the finite certificates. In particular, the D4/V4 and selected-atlas statements are supported by the explicit finite statements, tables, and replay artifacts in this package.

The nine bibliography entries have these roles:

| Key | Role |
|---|---|
| `BjornerBrenti2005` | Coxeter-group and symmetric-group background |
| `Stanley2012EC1` | finite-poset and linear-extension terminology |
| `Trotter1992` | partially ordered set background |
| `Stanley1986` | order-polytope context only; no order-polytope theorem is claimed |
| `BrightwellWinkler1991` | counting linear extensions context only |
| `PruesseRuskey1994` | generating linear extensions context only |
| `BjornerWachs1988` | generalized Coxeter quotient context only |
| `MartinSagan2006` | permutation-set transitivity context only |
| `BabanskyyTypeAPoset2026` | related-method context for type-A poset-cone work only |

The finite artifacts in `results/` and `certified/` are included so readers can inspect the exact data used by the paper.
