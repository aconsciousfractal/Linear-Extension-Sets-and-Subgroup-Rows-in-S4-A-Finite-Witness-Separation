# Public Package Manifest

- `paper/`: LaTeX source and compiled PDF.
- `results/`: finite JSON/CSV artifacts used by the paper.
- `certified/`: finite certificates and replay hash manifest.
- `tables/`: selected witness table used for the comparison proposition.
- `scripts/`: public replay and package-check scripts.
- `tests/`: minimal pytest checks for the public package.
- `docs/`: public claim and replay boundary notes.

## Hash Manifest Policy

`MANIFEST_SHA256.txt` hashes shipped repository files except itself, TeX temporary build files, cache directories, and `certified/PUBLIC_REPOSITORY_CERTIFICATION.json`. The certification JSON is generated by the package checker and is verified separately by `python -B scripts/check_public_package.py --check`.
